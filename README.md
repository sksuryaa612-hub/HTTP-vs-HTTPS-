# 🌐 HTTP vs HTTPS

## 📌 Overview
HTTP and HTTPS are protocols used for communication between a web browser and a server. The key difference between them is **security**, where HTTPS provides encrypted and secure data transfer.

---

## ⚖️ Comparison

| HTTP 🌐 | HTTPS 🔒 |
|--------|----------|
| HyperText Transfer Protocol | HyperText Transfer Protocol Secure |
| No encryption | SSL/TLS encryption enabled |
| Less secure | Highly secure |
| Port 80 | Port 443 |
| Data sent in plain text | Data is encrypted |
| http:// | https:// |

---

## 📝 Conclusion
HTTP is suitable for basic web communication, whereas HTTPS is recommended for secure data transmission, especially for login, payment, and sensitive information.
